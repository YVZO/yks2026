# 2026 YKS Kontenjan & Taban Puan Rehberi

ÖSYM'nin 2026 YKS Yükseköğretim Programları ve Kontenjanları Kılavuzu'nu PDF taramadan
arayıp filtrelemek için tek sayfalık web aracı — **21.482 program, 231 üniversite**.
Tablo 3 (ön lisans) + Tablo 4 (lisans). Başarı sırası ve taban puanı 2025 yerleştirme sonuçlarıdır.

## Kullanım
- **Çevrimiçi:** siteyi aç (index.html) — arama + Üniversite / Fakülte / Program kademeli filtreleri, puan türü / kurum / düzey filtreleri, maks. 2025 sırası, sütun sıralama.
- **Kendine indir:** sitedeki "İndir" satırından `YKS_2026_Kontenjan_Explorer.html` (aynı arayüz, tamamen çevrimdışı çalışır), `YKS_2026_Kontenjan.xlsx` (tam veri) veya kaynak PDF.

## Dosyalar
- `index.html` — site (açılışta `data.json` yükler)
- `data.json` — tam veri seti (~2,1 MB)
- `YKS_2026_Kontenjan_Explorer.html` — çevrimdışı tek dosya sürüm (veri gömülü)
- `YKS_2026_Kontenjan.xlsx` — Excel (Lisans / Ön Lisans sayfaları)
- `kontkilavuz_yktd21072026.pdf` — kaynak ÖSYM PDF (21 Temmuz 2026, ön sürüm)

Bu araç ÖSYM'nin resmi bir ürünü değildir. Kritik tercihleri güncel kılavuzla doğrulayın: https://www.osym.gov.tr/TR,33849/2026.html
