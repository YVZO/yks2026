# 2026 YKS Program & Quota Explorer

Searchable explorer for the OSYM "2026 YKS Yuksekogretim Programlari ve Kontenjanlari Kilavuzu"
(21 July 2026, preliminary edition) — 21,482 programs across 231 institutions.

- **Use online:** open the site (index.html) — search + University/Faculty/Program filters.
- **Take your own copy:** download `YKS_2026_Kontenjan_Explorer.html` (same UI, works fully
  offline in any modern browser) or `YKS_2026_Kontenjan.xlsx` (full dataset, two sheets:
  Lisans / On Lisans).

Rank and minimum score columns are the 2025 placement results published next to the 2026 quotas.
Data was extracted from the official PDF; verify critical choices against the current OSYM guide.
